A round
----

The next letter is revealed.

Everyone has one minute to type anagrams or pass.

Winning anagrams are resolved in time order. Note that multiple
anagrams are possible in one round, for example if two words that do
not share letters are available, or if a single person makes a
sequence of two anagrams.

* If at any time a person's sequence becomes unplayable due to
  anagrams formed by others, it is withdrawn.

* The last person to form a word or the first to pass "wins" the
  round.
